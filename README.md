# Panda Ar-Condicionado

Site institucional da empresa **Panda Ar-Condicionado**, especializada em climatização de ambientes residenciais, comerciais e industriais.

## 🚀 Sobre o projeto
Este site foi desenvolvido para apresentar a empresa, seus serviços e projetos, além de oferecer um canal de contato direto com clientes.

### ✨ Funcionalidades
- Página inicial com informações institucionais
- Seções: **Sobre**, **Serviços**, **Projetos** e **Contato**
- Botão de alternância para **Modo Escuro**
- Estrutura organizada em HTML, CSS e JavaScript
- Pastas dedicadas para imagens e vídeos

### 📂 Estrutura de pastas
index.html /css/style.css /js/script.js /imagens /videos

### 🛠 Tecnologias utilizadas
- **HTML5** para estrutura
- **CSS3** para estilização
- **JavaScript** para interatividade
- Hospedagem inicial via **Replit**, com integração futura ao **GitHub Pages** ou **Vercel**

## 📌 Objetivo
Fornecer uma presença digital profissional para a empresa Panda Ar-Condicionado, destacando seus serviços e facilitando o contato com clientes.
