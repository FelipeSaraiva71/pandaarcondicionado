# 🐼 Panda Ar-Condicionado

Site institucional da empresa **Panda Ar-Condicionado**, especializada em climatização de ambientes residenciais, comerciais e industriais.

## 🚀 Sobre o projeto
Este site foi desenvolvido para apresentar a empresa, seus serviços e projetos, além de oferecer um canal de contato direto com clientes.  
O layout foi pensado para ser moderno, responsivo e com animações suaves que valorizam a experiência do usuário.

### ✨ Funcionalidades
- **Hero Section** com logo, título animado e botão de contato via WhatsApp
- **Sobre**: apresentação institucional com destaque para experiência e diferenciais
- **Serviços**: cards interativos com hover e botão CTA
- **Resultados**: contadores animados e timeline visual
- **Empresas**: carrossel de logos com animação em cascata
- **Depoimentos**: grid responsivo com cards estilizados
- **Contato**: lista de informações e botão CTA com destaque
- **Animações globais**: efeitos `fadeUp`, `fadeDown`, `pulseScale` e `shine`
- **Responsividade**: ajustes específicos para mobile e tablet
- **Modo Escuro** (Dark Mode) com alternância via botão

### 📂 Estrutura de pastas
index.html
/css/style.css
/js/script.js
/imagens


### 🛠 Tecnologias utilizadas
- **HTML5** → estrutura semântica
- **CSS3** → estilização avançada com variáveis, animações e media queries
- **JavaScript** → interatividade (IntersectionObserver, contadores, toggle de menu e dark mode)
- **GitHub Pages** → hospedagem e publicação

### 🎨 Destaques de estilo
- Uso de **variáveis globais** para cores e transições
- **Animações personalizadas**: fade, pulse, reflexo em texto e carrossel em cascata
- **Layout mobile-first** com ajustes progressivos para tablets e desktops
- **Hover effects** em botões e cards para maior interatividade

## 📌 Objetivo
Fornecer uma presença digital profissional para a empresa **Panda Ar-Condicionado**, destacando seus serviços e facilitando o contato com clientes.
